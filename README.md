# ListPB
A small, customizable mod that displays your PBs next to each level in the Job Archive.

<img width="500" alt="ListPB Showcase" src="https://github.com/user-attachments/assets/02f82ff0-64f0-4a2c-8bd1-ce50d629560b" /><br />

## Installation

1. Install [NeonLite](https://github.com/Faustas156/NeonLite) by following it's installation instructions.
2. Download [the latest ListPB release](https://github.com/joeyexists/ListPB/releases/latest) and drop the `ListPB.dll` into your `Neon White\Mods` folder.
4. Press `F5` in-game to open melon preferences and change the mod settings.
