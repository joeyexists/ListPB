# ListPB
A small Neon White mod that displays your PBs next to each level in the Job Archive.

<img width="500" alt="ListPB Showcase" src="https://github.com/user-attachments/assets/f4776a34-8a60-4614-a369-840cd72aeb25" /><br />

## Installation

1. Install [NeonLite](https://github.com/Faustas156/NeonLite) by following it's installation instructions.
2. Download [the latest ListPB release](https://github.com/joeyexists/ListPB/releases/latest) and place the `ListPB.dll` into your `Neon White\Mods` folder.
4. Press `F5` in-game to open melon preferences and change the mod settings.
